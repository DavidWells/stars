# opn-arn

Open an [ARN](https://docs.aws.amazon.com/general/latest/gr/aws-arns-and-namespaces.html).

## Installation

```
npm i opn-arn
```

## Usage

```
$ opn-arn arn:aws:s3:::bucket-name/key/of/object
```
