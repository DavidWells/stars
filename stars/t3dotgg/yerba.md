# Yerba

![assets/example.png](assets/example.png)

An Electron Monorepo Demo

Uses:

- Next.js
- Typescript
- Tailwind
- Turborepo
- Vite (for Electron builds)

## Getting Started

```bash
npm install
npm run dev
```
