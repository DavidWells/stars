calculator
==========

simple calculator app with reactjs. and soon tests with jest

## Dependencies

- node.js
- npm (installed with node)
- grunt
- grunt-cli
- jest-cli --save-dev

## Dev flow

Build:

    npm install
    grunt browserify

Run:

    npm start

(open at [http://localhost:3000](http://localhost:3000))

Test:

    npm test
