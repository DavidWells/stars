# Features

- Handling data changes
- Column resizing
- Column and row reordering
- Sticky rows and columns
- Keyboard shortcuts
- Spanned cells
- Range selection
- Fill handle
- Styled ranges
- Custom styling
- Custom cell types
- Customizable behaviors
- API hook that allows you to interact with a ReactGrid
