## Install

```bash
$ npm install -g netlify-docs
```

## Usage

```bash
$ netlify-docs --help

  Usage:
    $ netlify-docs              Deploy `netlify-docs`

  Options:
    -h, --help                  Show help options
    -v, --version               Show version
```

_Make sure you have [netlify-cli](https://github.com/netlify/netlify-cli)
installed._
